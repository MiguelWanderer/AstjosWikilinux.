# ⚙️ AstjosWikiLinux // Archivum Omnissiah

> *"La carne es débil, pero la máquina es inmortal. Y cuando el código es puro, el conocimiento fluye."*

**AstjosWikiLinux** es una interfaz de cogitador inspirada en el Adeptus Mechanicus (Warhammer 40,000) diseñada para archivar, consultar y preservar los ritos sagrados de administración de servidores Linux. 

Este proyecto utiliza React para simular una terminal CRT de la Forja de Marte, con efectos de fósforo naranja, líneas de escaneo (scanlines) y engranajes mecánicos en movimiento perpetuo, albergando engramas de conocimiento (documentación modular) sobre el despliegue y mantenimiento de servidores.

## 🛠️ Tecnologías de la Forja (Stack)

* **React (Vite):** El núcleo lógico del cogitador.
* **React Markdown:** Módulo traductor para decodificar los pergaminos sagrados (`.md`) y mostrarlos en la terminal.
* **CSS3 (Puro):** Responsable del ungüento visual: efectos de monitor CRT antiguo, parpadeo de texto (flicker), resplandor de fósforo naranja y scanlines.
* **SVG Dinámico:** Engranajes de fondo generados matemáticamente y animados con CSS puro para movimiento perpetuo.

## 📜 Ritos de Inicialización (Instalación Local)

Si deseas clonar este cogitador en tu propia forja local, recita estos cánticos en tu terminal:

1. **Clonar la bóveda de datos:**
   ```bash
   git clone https://github.com/MiguelWanderer/AstjosWikilinux.git
   ```
2. **Ingresar al santuario del proyecto:**
   ```bash
   cd AstjosWikilinux
   ```
3. **Instalar las dependencias (Ungüentos de Node):**
   ```bash
   npm install
   ```
4. **Despertar al Espíritu de la Máquina:**
   ```bash
   npm run dev
   ```
